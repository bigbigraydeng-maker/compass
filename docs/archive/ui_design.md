# Compass 首版 UI 结构草图 (MVP版本)

## 设计原则

1. **中文为主**：所有界面以中文为主要语言
2. **数据可视化明显**：使用图表、卡片等可视化方式展示数据
3. **金融终端感**：采用专业、冷静、数据驱动的设计风格
4. **移动端适配**：确保在手机和平板上也能良好显示
5. **微信传播友好**：页面设计考虑在微信中分享的体验

## 颜色方案

- **主色调**：深蓝色 (#1E3A8A) - 体现专业和信任
- **辅助色**：浅蓝色 (#3B82F6) - 用于按钮和链接
- **成功色**：绿色 (#10B981) - 用于价格上涨、正面指标
- **警告色**：黄色 (#F59E0B) - 用于提醒和注意事项
- **背景色**：白色 (#FFFFFF) 和浅灰色 (#F3F4F6)
- **文字色**：深灰色 (#1F2937) 和浅灰色 (#6B7280)

## 字体方案

- **标题**：思源黑体 Bold (18px-32px)
- **正文**：思源黑体 Regular (14px-16px)
- **数字**：Roboto Mono (用于价格、统计数据)
